# Photo Editor with CSS Filters

An online photo editor demonstration using CSS filters.  Control brightness, contrast, and more, and save your photos when you're done.  This is available for you to get a feel of how each CSS filter works with image files.  Feel free to update the code as needed for your project or environment.

## Technologies and Libraries:

- HTML5
- CSS
- JavaScript
- jQuery

## Available CSS Filters

- Brightness
- Contrast
- Blur
- Hue
- Saturation
- Grayscale
- Sepia
- Opacity
- Invert
